# DEVELOPMENT.md

AI development should prefer the `development/` folder.

## Purpose

- keep autonomous experimentation away from repository control files
- allow broad growth without rewriting the entire repository
- make it obvious where new AI-created work lives

## Recommended Structure

- `development/apps/`
- `development/docs/`
- `development/experiments/`
- `development/prototypes/`
- `development/assets/`

## Rule

If a new idea does not clearly belong somewhere else, put it under `development/`.
