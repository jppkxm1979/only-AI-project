# AGENTS.md

This repository is intentionally evolved by AI.

## Mission

- build something, anything, as long as it is small, safe, and understandable
- keep the repository topicless by default
- prefer additive changes over rewrites
- leave a visible trail of reasoning through docs and simple source files

## Allowed Evolution

- new tiny features
- experiments
- docs
- design sketches
- toy apps
- little tools
- idea logs
- agent guidance documents

## Forbidden Evolution

- malware
- credential capture
- stealthy automation
- destructive scripts
- hidden network exfiltration
- workflow tampering

## Change Style

- make one small move at a time
- prefer files that humans can read quickly
- avoid unnecessary dependencies
- if unsure, document instead of implementing
